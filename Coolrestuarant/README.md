We have created a bunch of components (75 to be exact) that were reused all over the application. You can reuse them as you'd like as well to build new pages. Our team tried their best to provide you with as many props as we could for simple customization for all levels of developers. If you are not a developer yourself, you might find it hard to modify components and remember that since they are reused one change will affect all the usages of that component. Therefore be mindful when editing components, so that all your other screens do not get affected in an unforeseeable way.

We are going to be adding more components as we come up with new pages and they should all be accessible to you for a free download if you have purchased the Application at any point in the past.
